# Reports / Export Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add CSV and PDF export buttons to the Incidents and Threat Intelligence pages, exporting the currently-filtered data.

**Architecture:** Frontend-only, no backend changes. A shared utility file generates CSV blobs and PDF documents. Both pages import from it. Export respects the active filter state (`filtered()` / `filteredRows()`). A small dropdown (signal-driven) lets the user pick CSV or PDF.

**Tech Stack:** Angular 19 signals, `jspdf`, `jspdf-autotable`, native Blob/URL API.

## Task 2 — Shared export utilities

**File:** Create `frontend/src/app/core/utils/export.utils.ts`

- [ ] **Step 1: Create the file**

```typescript
import jsPDF from 'jspdf';
import autoTable from 'jspdf-autotable';

export function downloadCsv(headers: string[], rows: string[][], filename: string): void {
  const lines = [headers, ...rows].map(
    row => row.map(cell => `"${String(cell ?? '').replace(/"/g, '""')}"`).join(',')
  );
  const blob = new Blob([lines.join('\n')], { type: 'text/csv;charset=utf-8;' });
  const url  = URL.createObjectURL(blob);
  const a    = document.createElement('a');
  a.href     = url;
  a.download = filename;
  a.click();
  URL.revokeObjectURL(url);
}

export function downloadPdf(title: string, headers: string[], rows: string[][], filename: string): void {
  const doc = new jsPDF();
  doc.setFontSize(13);
  doc.setTextColor(40);
  doc.text(title, 14, 15);
  doc.setFontSize(8);
  doc.setTextColor(130);
  doc.text(`Generated ${new Date().toLocaleString()}`, 14, 22);
  autoTable(doc, {
    head:       [headers],
    body:       rows,
    startY:     28,
    styles:     { fontSize: 8, cellPadding: 3 },
    headStyles: { fillColor: [30, 37, 53], textColor: [229, 231, 235], fontStyle: 'bold' },
    alternateRowStyles: { fillColor: [245, 247, 250] },
  });
  doc.save(filename);
}
```

Add signal and methods inside the `Incidents` class (after the `filtered` computed):
```typescript
  exportOpen = signal(false);

  private get exportRows(): string[][] {
    return this.filtered().map(i => [
      i.id,
      i.title,
      i.severity,
      i.status,
      i.attack_type,
      i.source_region,
      String(i.event_count),
      i.assigned_to ?? '—',
      new Date(i.created_at).toLocaleString(),
    ]);
  }

  private readonly exportHeaders = ['ID', 'Title', 'Severity', 'Status', 'Type', 'Region', 'Events', 'Assigned', 'Created'];

  exportCsv() {
    downloadCsv(this.exportHeaders, this.exportRows, 'incidents.csv');
    this.exportOpen.set(false);
  }

  exportPdf() {
    downloadPdf('SignalForge — Incident Report', this.exportHeaders, this.exportRows, 'incidents.pdf');
    this.exportOpen.set(false);
  }
```
